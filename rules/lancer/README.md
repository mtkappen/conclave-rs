# Lancer Rules - File Structure Overview

This folder contains the Lancer Free Version rulebook broken down into organized markdown files by section.

## Directory Structure

```
lancer/
├── getting-started/
│   ├── 0-introduction.md           # Title page, credits, content warnings
│   ├── 1-playing-lancer.md         # Core rules, dice mechanics, bonuses
│   └── 2-missions-downtime-scenes.md
│
├── building-pilots-and-mechs/
│   ├── 0-license-levels.md         # LL progression, level chart
│   ├── 1-the-pilot-backgrounds.md  # Background options and descriptions
│   ├── 2-triggers.md               # All trigger options and usage
│   └── 3-the-mech-structure.md     # Frames, mounts, weapons, systems
│
├── missions-uptime-downtime/
│   └── (to be created)
│
├── mech-combat/
│   └── (to be created)
│

    └── (to be created)
```
