# FALSE VISION

## Spell 5 - Uncommon Illusion

**Traditions:** arcane, occult  
**Cast:** [two-actions] somatic, verbal  
**Range:** touch; **Targets:** 1 object or location  
**Duration:** 24 hours  

You mask the true nature of a place or object with an illusion. You can make it appear to be anything you wish, including empty space, a different location, or a mundane object. Creatures that interact with the false vision can attempt a Will save to disbelieve.
