# Pathfinder 2e Core Rulebook

This folder contains the organized rules from the Pathfinder Second Edition Core Rulebook, broken down into logical sections and markdown files for easy reference and integration.

## Structure

```
Pathfinder-2e-core-Rulebook/
├── 01-introduction/          # Chapter 1: Getting Started
│   ├── 01-overview.md        # What is Pathfinder, RPG basics
│   ├── 04-key-terms.md       # Key terminology and definitions
│   ├── 05-example-of-play.md # Example of play walkthrough
│   └── 06-character-creation.md # Character creation guide
├── 02-ancestries-backgrounds/ # Chapter 2: Character Origins
│   ├── 01-overview.md        # Ancestry and background introduction
│   ├── 02-dwarf.md           # Dwarf ancestry, heritages, feats
│   ├── 03-elf.md             # Elf ancestry, heritages, feats
│   ├── 04-gnome.md           # Gnome ancestry, heritages, feats
│   ├── 05-goblin.md          # Goblin ancestry, heritages, feats
│   ├── 06-halfling.md        # Halfling ancestry, heritages, feats
│   ├── 07-human.md           # Human ancestry, heritages, feats
│   └── 08-backgrounds.md     # Background options and languages
├── 03-classes/               # Chapter 3: Character Classes (In Progress)
├── 04-skills/                # Chapter 4: Skills
├── 05-feats/                 # Chapter 5: Feats
├── 06-equipment/             # Chapter 6: Equipment
├── 07-spells/                # Chapter 7: Spells
├── 08-age-of-lost-omens/     # Chapter 8: Setting
├── 09-playing-the-game/      # Chapter 9: Core Rules
├── 10-game-mastering/        # Chapter 10: GM Advice
├── 11-crafting-treasure/     # Chapter 11: Items and Treasure
└── appendices/               # Conditions, character sheets, glossary
```

## Completed Sections

### Chapter 1: Introduction ✅
- Complete overview of Pathfinder RPG basics
- Key terms and definitions glossary  
- Full example of play combat scenario
- Step-by-step character creation guide

### Chapter 2: Ancestries & Backgrounds ✅
- All 6 core ancestries with heritages and feats:
  - Dwarf (5 heritages, 10+ feats)
  - Elf (5 heritages, 9+ feats)
  - Gnome (5 heritages, 10+ feats)
  - Goblin (5 heritages, 10+ feats)
  - Halfling (5 heritages, 10+ feats)
  - Human (4 heritages including Half-Elf/Half-Orc, 20+ feats)
- Complete backgrounds section with all 30+ background options

## Usage

Each markdown file is organized by section with clear headers for easy navigation. Rules elements are formatted consistently for readability.
