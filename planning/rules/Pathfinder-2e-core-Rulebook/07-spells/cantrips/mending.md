# Mending

**Level:** Cantrip
**Traditions:** arcane, divine, primal  
**Cast:** [two-actions] somatic, verbal  
**Range:** touch  
**Targets:** 1 object of 1 Bulk or less  
**Duration:** see below  

You repair a single object. If the target is broken, it repairs 2d6 Hit Points to the object. If this brings the object's HP above its Broken Threshold, the object is no longer broken.

If the target is damaged but not broken, you restore 1d8+4 Hit Points to it. You can't repair an object beyond its maximum HP.
**Heightened (+2):** The healing increases by 1d6 (for broken objects) or 1d4+2 (for damaged objects).
