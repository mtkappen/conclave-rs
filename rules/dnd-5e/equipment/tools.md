# Tools

Tools represent specialized equipment used for specific crafts, games, or activities. Proficiency with a tool allows you to add your Proficiency Bonus to ability checks made using that tool.

## Tool Categories

### Artisan's Tools
These tools are used by craftsmen to create goods and items:
- **Alchemist's Supplies:** Used to create potions, acids, and other chemical substances
- **Brewer's Supplies:** Used to brew beer and ale
- **Calligrapher's Supplies:** Used for elegant writing and illumination
- **Carpenter's Tools:** Used to work with wood
- **Cartographer's Tools:** Used to create maps
- **Cobbler's Tools:** Used to make and repair shoes
- **Cook's Utensils:** Used to prepare food
- **Glassblower's Tools:** Used to shape glass
- **Jeweler's Tools:** Used to work with precious stones and metals
- **Leatherworker's Tools:** Used to work with leather
- **Mason's Tools:** Used to work with stone
- **Painter's Supplies:** Used for painting
- **Potter's Tools:** Used to shape clay
- **Smith's Tools:** Includes blacksmith's tools used to forge metal
- **Tinker's Tools:** Used to repair and create small mechanical devices
- **Weaver's Tools:** Used to work with fabric
- **Woodcarver's Tools:** Used for detailed wood carving

### Gaming Sets
These are used for various games of chance:
- **Dice Set:** Used for games like Dragonchess and other dice-based games
- **Playing Card Set:** Standard playing cards for various games
- **Three-Dragon Ante:** A popular card game

### Musical Instruments
Used by Bards and other performers:
- **Bagpipes**
- **Drum**
- **Dulcimer**
- **Flute**
- **Lute**
- **Lyre**
- **Horn**
- **Pan Flute**
- **Shawm**
- **Viol**

### Specialized Tools
- **Disguise Kit:** Used to create false appearances and disguises
- **Forgery Kit:** Used to create forged documents and items
- **Herbalism Kit:** Used to identify plants, make poultices, and prepare herbal medicines (Druids gain proficiency)
- **Navigator's Tools:** Used to navigate by stars or maps
- **Poisoner's Kit:** Used to create and apply poisons
- **Thieves' Tools:** Used to disarm traps and pick locks (Rogues gain proficiency)

## Tool Proficiency Benefits

When you have proficiency with a tool:
1. You add your Proficiency Bonus to ability checks made using that tool
2. Some tools allow you to create items or perform special tasks
3. Certain class features require specific tool proficiencies

### Creating Items
Proficiency with artisan's tools allows you to craft nonmagical items of that type. The time and cost depend on the item's complexity.

### Tool Checks
DMs call for tool checks when:
- Crafting or repairing items
- Identifying substances (herbs, poisons, etc.)
- Creating disguises or forgeries
- Navigating through difficult terrain
- Disarming traps or picking locks

## Tool Combinations

Some activities may require multiple tools. For example:
- Creating a forged document might require both a Forgery Kit and Calligrapher's Supplies
- Brewing a potion requires Alchemist's Supplies and possibly Cook's Utensils
- Crafting a magic item typically requires specialized tools plus spellcasting ability
