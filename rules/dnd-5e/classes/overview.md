# Class Overview

This section contains detailed information about each of the twelve character classes in D&D 5e. Each class represents a distinct path of adventure and combat specialization.

## Quick Reference

| Class | Primary Ability | Hit Die | Key Features |
|-------|----------------|---------|--------------|
| Barbarian | Strength | d12 | Rage, Unarmored Defense, Extra Attack |
| Bard | Charisma | d8 | Bardic Inspiration, Spellcasting, Expertise |
| Cleric | Wisdom | d8 | Spellcasting, Channel Divinity, Domain Features |
| Druid | Wisdom | d8 | Spellcasting, Wild Shape, Nature Magic |
| Fighter | Strength/Dexterity | d10 | Fighting Style, Second Wind, Action Surge, Extra Attack |
| Monk | Dexterity/Wisdom | d8 | Martial Arts, Ki Points, Unarmored Movement |
| Paladin | Strength/Charisma | d10 | Divine Sense, Lay on Hands, Spellcasting, Smite |
| Ranger | Dexterity/Wisdom | d10 | Spellcasting, Primeval Awareness, Extra Attack |
| Rogue | Dexterity | d8 | Sneak Attack, Cunning Action, Expertise |
| Sorcerer | Charisma | d6 | Spellcasting, Metamagic, Sorcerous Origin |
| Warlock | Charisma | d8 | Pact Magic, Eldritch Invocations, Patron |
| Wizard | Intelligence | d6 | Spellcasting, Arcane Recovery, School Specialization |

## Choosing a Class

When choosing a class, consider:
- **Primary Ability:** Which ability score is most important for the class?
- **Hit Die:** How durable do you want your character to be?
- **Play Style:** Do you prefer melee combat, ranged attacks, spellcasting, or skill-based gameplay?
- **Complexity:** Some classes (like Wizard and Bard) have more options and require more management than others (like Fighter or Rogue)

## Multiclassing

You can multiclass to combine features from multiple classes. See the Character Creation section for multiclassing rules and prerequisites.
